# firm/ — Firmen-Landkarte (leeres Skelett)

> Der eine Einstieg fuer Firmen-Wissen ueber die AI-Firma. Aktuell leer — jeder Fakt bekommt EINE
> Stelle, der Rest verlinkt dorthin (kein Fakt zweimal). Nummern-Schema abgeschaut vom swisscare-brain.

| Schicht | Was hier reinkommt |
|---|---|
| `00-fakten/` | Die eine Wahrheit pro Fakt (Firmenname, Ziele, Preis-Modell, harte Regeln). |
| `10-team/` | Wer macht was (aktuell SA solo; ggf. spaeter Helfer). |
| `20-prozesse/` | Ablaeufe: Neukunde Phase 0-5, Angebot, Abnahme, Betrieb. |
| `30-recht/` | Rechtlich Bindendes: Werkvertrag-Bausteine, Haftungs-Deckel, Wiederverwendung. |
| `40-strategie/` | Ziele, USP, Nordstern — wohin die Firma will. |
| `50-systeme/` | Werkzeuge: Repo, Vorlagen, Kunden-Repos, Brain. |
